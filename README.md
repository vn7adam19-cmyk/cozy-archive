# Cozy

A web app built with [TanStack Start](https://tanstack.com/start) (React) and [Supabase](https://supabase.com), originally built with [Lovable](https://lovable.dev).

## Tech stack

- React + TanStack Start (routing, SSR)
- TypeScript
- Tailwind CSS + Radix UI
- Supabase (auth & database)

## Getting started

1. Clone the repo and install dependencies:

   ```bash
   npm install
   ```

2. Copy `.env.example` to `.env` and fill in your Supabase project details:

   ```bash
   cp .env.example .env
   ```

3. Run the dev server:

   ```bash
   npm run dev
   ```

## Build

```bash
npm run build
```
